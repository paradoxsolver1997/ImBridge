ImBridge Help

ImBridge is a local tool for batch image/document conversion and enhancement. It supports conversion between various bitmap and vector formats, resolution enhancement, grayscale/binarization, and batch processing. Suitable for research, office, printing, and more.

Main Features

  • Batch image format conversion: Supports conversion between PNG, JPEG, BMP, TIFF, SVG, PDF, EPS, PS, etc.
  • Vector and bitmap conversion: High-quality conversion between vector (SVG/PDF/EPS/PS) and bitmap (PNG/JPG/BMP/TIFF) formats.
  • Resolution enhancement: Improve image resolution using interpolation and sharpening algorithms, suitable for printing and high-precision needs.
  • Grayscale/Binarization: Support for grayscale and binarization, suitable for document scanning, signatures, etc.
  • Batch processing: Batch conversion and enhancement for multiple files, improving efficiency.
  • Vector analysis: Automatically analyze PDF/SVG/EPS vector files, distinguish vector/raster/mixed types.

Typical Workflows

    Bitmap conversion

    To convert between bitmap formats (PNG, JPEG, BMP, TIFF) or to vector formats (SVG, PDF, EPS, PS):

    1. Navigate to the Bitmap tab.
    2. Input/Output Settings:
      • Select input files (multi-select supported)
      • Choose output folder.
      • Check the input image in Preview frame
    3. Option 1. Convert to Bitmaps (batch convert to other bitmap formats):
      • Select Output Format (PNG, JPEG, BMP, TIFF)
      • Set Quality parameter (for .jpg/.jpeg only).
      • Click Convert to batch convert formats automatically.
    4. Option 2. Convert to Vectors (batch embed bitmaps into vectors):
      • Select Output Format (SVG, PDF, EPS, PS)
      • Set DPI parameter (not applicable to .svg).
      • Click Convert to batch convert formats automatically.

    Vector and bitmap conversion

    To convert between vector formats (SVG, PDF, EPS, PS) or to bitmap formats (PNG, JPEG, BMP, TIFF):

    1. Navigate to the Vector tab.
    2. Input/Output Settings:
      • Select input files (multi-select supported)
      • Choose output folder.
      • Check the input image in Preview frame
    3. Tool Check:
      • ImBridge relies on the following Python dependencies to process SVG and PDF files. They are already included in requirements.txt. No need to install them separately.
        • cairosvg: A Python library for converting SVG files to PDF, PNG, and other formats.
        • PyPDF2: A Python library for reading and analyzing PDF files.
      • Note that cairosvg requires libcairo-2.dll which should be installed independently. Install it by following https://www.cairographics.org/download/. However, things are easier if you have already installed GIMP, an open-source image processing software. Simply add GIMP_DIR/bin to your $PATH environment variable.
      • The following external tools are also needed to enable vector analysis or conversion. They should be installed and configured separately. Follow the links above to install the required tools. Make sure they are in the $PATH environment variable.
        • Ghostscript: A powerful interpreter for PostScript and PDF files, used for vector/bitmap conversion. Download it from https://www.ghostscript.com/
        • pstoedit: A tool to convert PostScript and PDF graphics to other vector formats such as SVG. Download it from http://www.pstoedit.net/
      • The Tool Check section in the ImBridge interface will indicate whether these tools are installed and accessible.
    4. Option 1. Analyze Vectors:
      • Click Analyze button to analyze the selected vector files.
      • Check Log Frame for analysis results.
    5. Option 2. Convert to Vectors:
      • Select Output Format (SVG, PDF, EPS, PS)
      • Click Convert to batch convert formats automatically.
    6. Option 3. Convert to Bitmaps:
      • Select Output Format (PNG, JPEG, TIFF)
      • Set DPI parameter.
      • Click Convert to batch convert formats automatically.

    Resolution enhancement & grayscale

    To enhance image resolution or convert to grayscale/binary images:

    1. Navigate to the enhancement tab.
    2. Input/Output Settings:
      • Select input files (multi-select supported)
      • Choose output folder.
      • Check the input image in Preview frame
    3. Option 1. Upscaling
      • Set Scale Factor.
      • Click Upscale & Save for batch processing.
    4. Option 2. Grayscale
      • Click Grayscale & Save for batch processing.
    5. Option 3. Vectorization (Use Potrace to convert images to vector format.)
      • Set Output Format.
      • Click Vectorize & Save for batch processing.
    6. Tool Check
      • ImBridge relies on Potrace to enable vectorization. It should be installed and configured separately. Follow the links above to install the required tools. Make sure the executable is in the $PATH environment variable.
        • Potrace: A utility for tracing bitmaps and converting them into smooth, scalable vector graphics. Download it from http://potrace.sourceforge.net/

FAQ & Troubleshooting

  • Q: Why is the converted image blurry or the size incorrect?
    Check DPI settings and output format. For vector to bitmap, set an appropriate resolution.
  • Q: Why can't some formats be converted?
    Make sure Ghostscript, pstoedit, Potrace, etc. are installed and in your PATH.
  • Q: How to batch process many files?
    You can multi-select input files, or drag files into the input box for automatic batch processing.

Security & Privacy

  • All image processing is done locally, nothing is uploaded to the cloud, ensuring data security.
